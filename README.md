# Linktree

Desafio de criar uma página apresentando links pessoais semelhante a um linktree

## Tecnologias

- HTML
- CSS
- JavaScript
- React

## Escopo do projeto

### Home

Página onde o usuário pode visualizar os diferentes links pessoais
